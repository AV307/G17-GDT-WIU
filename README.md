Starting Keys:
0 - Proceed

Movement Keys:
W - Forward
A - Left
S - Backward
D - Right

Skill Tree Keys:
/ - Open
1 - First Stat
2 - Second Stat
3 - Third Stat
Any Other Key - Force Exit

While In Combat Keys:
F - Fight
I - Item
R - Run
Y - Yes (if confirmation is required)
N - No (if confirmation is required)
0 to 8 - Select Category/Index (if Item Action is in play)

Playing Keys:
Space - Interact
